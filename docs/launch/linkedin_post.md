Excited to share ProjectOS, a project I built during my final year at IIT Madras!

As a student balancing coursework, projects, and learning new stacks, I struggled with constant context switching. That’s why I built ProjectOS: an autonomous multi-agent assistant that runs silently in the background of your code repository.

When you feed it a high-level feature request, ProjectOS orchestrates 7 specialized AI agents to plan the changes, write the code, run review checks, write unit tests, and update documentation. It uses quality gates to grade code before any file is updated, ensuring bad code never reaches your files.

Technical Stack:
- Python
- Google Gemini (runs fully on the free tier)
- Local Vector RAG & Semantic Routing

It is 100% open-source, runs locally on your machine, and preserves complete data privacy. Check out the setup and start running your background team today!
