# Project Vision

## Project

ProjectOS is a Personal Engineering OS for Aryan Kumar's projects.

## Classification

ProjectOS is an autonomous multi-agent system for handling the full
software development life cycle for Aryan Kumar's projects without
requiring Aryan to context-switch between engineering roles.

## Current User

The current user is only Aryan Kumar.

## Long-Term Direction

The long-term direction is:

- Open-source
- Contributor community
- Potential product

## Operating Context

Aryan must balance:

- IIT Madras final year academic focus
- Project development, including content creation pipeline work and data
  science projects
- Startup ideation

## Desired Shift

Current AI tools are reactive: they do what they are asked once.

ProjectOS must be proactive: it must run continuously without Aryan.

## Launch Scope

ProjectOS is for Aryan's personal projects at launch.

The launch deployment is a single-machine deployment.

## Version 1 Scope Boundary

Version 1 excludes:

- Multi-user support
- Web dashboard
- Self-modifying agents
- Enterprise features
- Non-Python projects at launch
