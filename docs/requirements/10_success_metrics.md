# Success Metrics

## 30-Day Minimum Success Definition

Within 30 days, these must work end-to-end:

- Clone
- Planning Agent
- Code Review Agent

## 30-Day Stretch Success Definition

Within 30 days, the stretch definition is:

- All five agents scaffolded
- Three agents fully functional

## 30-Day Must-Haves

Within 30 days, these are must-have outcomes:

- `/model` switch working
- Daemon running in the background

## Required Agents Over Time

All five agents are required:

- Planning Agent
- Code Review Agent
- Architecture Agent
- Test Agent
- Documentation Agent

Phased delivery is acceptable.
